## Features

- Disable autoplay
- Hide suggested images and videos
- Disable infinity scrolling
- Disable reels

<br/>

### Get started

1. Install Node.js
2. Install Git LFS ( for the assets of the internal test )
3. Run `pnpm install`
4. Run `pnpm dev`
